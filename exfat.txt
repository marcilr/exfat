exFAT


The exFAT file system is ideal for flash drives and SD cards.
It’s like FAT32, but without the 4 GB file size limit.  You
can use exFAT drives on Linux with full read-write support,
but you’ll need to install a few packages first.
--www.howtogeek.com/235655/how-to-mount-and-use-an-exfat-drive-on-linux/
